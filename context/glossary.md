# Glossary

| Term | Definition |
|---|---|
| Product | Medical equipment available for rent (previously "Equipment listing") |
| Merchant | Equipment supplier with a storefront (previously "Supplier") |
| Buyer | Healthcare facility that browses, carts, and rents equipment |
| Cart Line Item | A product in the cart with rental period and quantity |
| Order | Confirmed rental containing one or more sub-orders |
| Sub-Order | Per-merchant split of an order, each with own payment/contract |
| Rental Period | Start date + end date for equipment rental |
| Fulfillment Status | dispatched → in_transit → delivered → returned |
| Payment Status | pending → completed / failed / expired |
| Snippe | Payment processor for mobile money (M-Pesa, Airtel, etc.) |
| SendAfrica | SMS layer for order confirmations and notifications |
| Merchant Trust Card | Component showing merchant name, verification badge |
